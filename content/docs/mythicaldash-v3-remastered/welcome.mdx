# Welcome to MythicalDash V3 Remastered

MythicalDash is a powerful client area control dashboard that allows users to view and manage their server resources seamlessly. With customizable RAM, Disk, and CPU limits, users can create and control as many servers as allocated through an intuitive interface.

The dashboard integrates with Pterodactyl as the game panel backend, utilizing its robust API to create servers and manage users efficiently.
